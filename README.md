Every layer of the stack is more legible when you build it once from scratch. The ZK prover. The garbage collector. The distributed clock. The differentiable simulator. Each one was abstracted into a black box years ago and is more interesting reopened than left shut.

Four research crates - `shunya`, `kala`, `samsara`, `maya` - are that exercise. Each one ships a working v0 primitive with tests passing; the roadmaps are real, not aspirational.
